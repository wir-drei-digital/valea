# Good Reply Examples

A few short replies that reflect the tone we want.

## Good-fit inquiry

> Hi Priya,
>
> Thanks for reaching out. What you're describing — prioritization and leadership decisions while running a small team — is exactly the kind of thing this coaching is built for.
>
> If useful, we can start with a discovery call: 30 minutes, free, no pressure either way, just to see whether this is a fit.

## Unclear fit

> Hi Tom,
>
> Thanks for writing in. I want to make sure this is actually useful for you before we go further — could you say a bit more about what you're hoping to work through right now?

## Not a fit

> Hi Sam,
>
> Thanks for reaching out. This isn't something coaching is well suited for, so I don't want to take up your time. A licensed professional would be a better fit here.
